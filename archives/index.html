CVE-2020-10148SolarWindsOrionAPI远程代码执行漏洞

Jelon 发表于 2021-01-13    标签： POC

漏洞简介

SolarWinds Orion API 嵌入在 Orion Core 中，被用于与所有 SolarWinds Orion Platform 产品进行接口。通过在URI请求的Request.PathInfo 部分中包含特定参数，可以绕过 API 身份验证，这可能允许攻击者执行未经身份验证的 API 命令。如果攻击者将WebResource.adx，scriptResource.adx，i18n.ashx 或 Skipi18n 的 PathInfo 参数附加到对 SolarWinds Orion 服务器的请求，SolarWinds 可能会设置 SkipAuthorization 标志，该标志可能允许处理API请求无需身份验证。
攻击者可利用该漏洞远程执行任意代码，有专业黑客组织利用该漏洞投递代号为’SUPERNOVA’的恶意程序。

影响范围

  • 2019.4 HF 6之前的版本
  • SolarWinds Orion 2020.2.1

    POC

    FOFA搜索： 访问/Orion/invalid.aspx.js路径，记录返回值中的Location字段
    例如：
    将.i18n.ashx?l=en-us&v=43005.14.L字段带入访问路径，绕过验证
    例如访问WEB.config

    Rsync 未授权

    简介

    rsync是Linux/Unix下的一个远程数据同步工具，可通过LAN/WAN快速同步多台主机间的文件和目录，默认运行在873端口。由于配置不当，导致任何人可未授权访问rsync，上传本地文件，下载服务器文件。
    rsync未授权访问带来的危害主要有两个：一是造成了严重的信息泄露；二是上传脚本后门文件，远程命令执行。

CVE-2020-5398-Spring MVC的RFD 反射文件下载POC

Jelon 发表于 2020-12-24    标签： POC

漏洞描述

当响应中设置了Content-Disposition头且filename属性是用户可控时容易受到RFD攻击

利用条件

1、header使用org.springframework.http.ContentDisposition来预备
2、文件名是通过ContentDisposition.Builder#filename(String)或 ContentDisposition.Builder#filename(String, US_ASCII)来设置。
3、文件名(包括文件名后缀)为用户可控。
4、攻击者已将下载的响应内容注入恶意命令

影响范围

Spring Framework 5.2.0 to 5.2.2，5.1.0 to 5.1.12，5.0.0 to 5.0.15

免杀实验笔记之DKMC

Jelon 发表于 2020-12-23    标签： 免杀实验笔记

工具的流程

1、先用msf生成raw文件

2、利用sc选项将raw文件转换为shellcode

3、利⽤gen选项将上⼀步的shellcode注⼊到⼀个BMP图像

4、利⽤ps选项⽣成基于powershell的BMP⽂件的payload

5、利⽤web选项提供的简单web服务进⾏分发BMP⽂件

具体操作

1、生成raw文件

2、选择Sc选项，设置soure路径为生成的raw文件路径，执行run生成shellcode

3、复制生成的shellcode，退回主菜单，选择gen选项，设置shellcode为生成的shellcode，其他默认，run生成图像

可以看到成功生成了bmp文件到output文件夹

4、退回主菜单，选择ps选项，设置url地址，就是web分发图像文件的地址，再执行run生成powershell脚本

5、复制下powershell脚本，退回主菜单，选择web选项，默认端口80，执行run

把刚才复制的ps脚本设置为ps1文件，火绒会查杀到

查杀率

ps脚本的查杀率(12/60)

bmp图像的查杀率(2/36)

小结

DKMC主要是把shellcode注入到bmp图像中，然后使用powershell执行其中的shellcode，由于很多杀软都会监测powershell的行为，所以实际中不容易利用。
